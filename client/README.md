# OpenDesk — VT Study Spot Finder (Client)

React + Vite frontend for the OpenDesk study spot finder app.

## Setup

```bash
npm install
npm run dev
```

Requires the API server running on `localhost:3001`. The dev server proxies `/api/*` requests automatically.

## Build

```bash
npm run build
```

## Stack

- React 18
- React Router v6
- Tailwind CSS v3
- Vite 5
